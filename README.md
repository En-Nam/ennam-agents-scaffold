# @ennam/agents-scaffold

Install Claude Code tooling (Superpowers + Serena + role agents + MCP) into any existing project.

```bash
npx @ennam/agents-scaffold <profile>
```

Profiles: `flutter`, `next`, `python`, `go`, `qa`. See [design spec](docs/spec.md) for details.

**Status:** MVP — only `next` profile available. Other profiles coming in v0.2.

## Quickstart

```bash
cd my-existing-nextjs-app
npx @ennam/agents-scaffold next
```
